# Разработка интернет-приложений (РИП) 2025

## Расчет стоимости лиценизрования ПО

Разработанное приложение позволит покупателям создавать и формировать заявки на расчёты лицензирования стоимости ПО для выбранных ими лицензий, а модераторам завершать или отклонять данные заявки.
